Whyteboard: A free painting application

Whyteboard is a free painting application that can run on Linux, Windows and Mac. It is suited towards creating visual presentations and for overlaying PDF images with annotations.

Features

  • Draw using familiar tools: pen, rectangle, circle, text, arrows
  • Annotate over PDF files
  • Resize, move and rotate shapes you've drawn
  • Multiple sheets - draw with an intuitive tab interface
  • Embed an audio/video player onto the canvas
  • Your drawings are recorded and can be replayed
  • Each sheet has a preview thumbnail of itself
  • Note items, similar to "sticky notes"
  • Resize the canvas easily by dragging anywhere outside of its border
  • Translated into many languages e.g. Chinese, French, German, Spanish, Italian, Turkish, Portuguese and many more
  • Closed sheets can be re-opened, restoring their drawings and complete undo and drawing history

Help Translate

  1. Register at Launchpad.net and set your preferred language.
  2. Go to Whyteboard Translations at Launchpadand click the language you intend to help with. You must set your preferred language to be able to translate into it.
  3. The translation interface is shown - each English message and an input box the translation.
  4. When translating, keep codes such as%s,%iand%dintact - these have special meaning in the program.
  5. The & symbols you see, e.g.&Fileand&Editare used asKeyboard Accesskeys.
  6. Try to Capitalize Menu Items and to use good grammar throughout.
  7. Your contribution will be added to the next release of Whyteboard and your name added to the credits!
  8. Learn more abouttranslation support on Launchpad.
