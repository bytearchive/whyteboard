Shape Viewer

Overview

The shape viewer allows you to see and manage the order, and properties of each sheet's drawn items.

Usage

From the View Menu, select Shape Viewer to bring a dialog displaying a list of all drawn items.
Items at the top of the list are drawn on top of other items below it.

The name of the drawn item is shown, as well as its thickness and colour (may be an RGB value).
The buttons can be used to move the items' position in the list: they are "move to top", "move up", "move down", "move to bottom".

Note on the Select tool/h4> When you select an item with the Select tool, it is brought to the front of the list temporarily
